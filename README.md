# Stream Companion

An AI Streamer Companion that allows integration with many streaming platforms and services through Websocket.

## Features

- Websocket server for real-time communication with streaming platforms and services.
- **COMING SOON** - Custom actions triggered by the AI model sending messages to a websocket server.

## Installation

You can follow the instructions in the [docs](https://stream-companion.olliedev.ca).
